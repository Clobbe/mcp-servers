📦 INSTALLATION (ONE-TIME SETUP)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Navigate to directory:
   cd ~/dev/tooling/mcp-servers

2. Install & build all servers:
   for server in prd-generator ralph-workflow changelog-manager code-tools; do
     cd $server && npm install && npm run build && cd ..
   done

3. Configure your AI platform:
   Edit ~/.claude/settings.json (or ~/.opencode/settings.json)
   
   Add absolute paths to each server's build/index.js
   Example: /Users/YOUR_USERNAME/dev/tooling/mcp-servers/prd-generator/build/index.js

4. Restart your AI platform

🐛 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Issue: "Server not found"
Fix:  Use absolute paths, restart AI platform

Issue: "Cannot find module"
Fix:  cd server-name && npm install && npm run build

Issue: "Tools not appearing"
Fix:  Check config JSON syntax, verify build succeeded

Issue: TypeScript errors
Fix:  cd server-name && rm -rf build && npm run build
